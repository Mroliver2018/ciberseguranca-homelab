# Blue Team / SOC Homelab

Este repositório documenta meu laboratório prático de cibersegurança focado em
Blue Team e SOC, utilizando VMware.

## Objetivos do laboratório
- Monitorar eventos de segurança em Windows, Linux e Active Directory
- Analisar logs e alertas em um SIEM (Wazuh)
- Praticar resposta a incidentes
- Aplicar conceitos do MITRE ATT&CK

## Ambiente
- Windows Server 2019 (Domain Controller)
- Windows 10 (cliente de domínio)
- Ubuntu Server (servidor Linux)
- Wazuh SIEM
- Rede Host-only (isolada)

## Principais atividades
- Análise de falhas de login
- Monitoramento de contas privilegiadas
- Detecção de acessos suspeitos
- Criação de playbooks de resposta a incidentes

Este laboratório é utilizado para fins educacionais.

Iniciando o projeto

SIEM WAZUH/

Instalação da VM Ubuntu Server 24.04, dentro desse SO foi configurado acesso via SSH e instalação pelo terminal o SIEM WAZUH, configurado para treinamento de monitorização de ataques e rede.
